# Space Flight News Api 

## Description
This project is a Space Flight News REST API built with NodeJs and MongoDB.
The purpose of this api is a challenge proposed by Coodesh.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
```
